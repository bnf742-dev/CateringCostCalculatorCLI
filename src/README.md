# Catering Cost Calculator CLI

A Java command-line application for catering portion cost analysis and full event quote generation.

## Features

- Portion cost calculator
- Target food cost pricing
- Event quote generator
- Sales tax and service charge calculation
- Modular method-based structure

## Technologies Used

- Java
- IntelliJ IDEA
- Git & GitHub

## How It Works

The application runs in the command line and allows the user to:

1. Calculate portion cost based on cost per pound and portion size.
2. Generate a full event quote based on per-person pricing, tax, and service charges.
